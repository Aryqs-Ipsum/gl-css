gl css

gl css is a material design framework inspired by new google rounded style. It includes lots of easy to use components to quickly develop web interfaces.

Quickstart

Use assets with CDN link :

<!-- Material Design icons -->
<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">

<!-- gl-css files -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Aryqs-Ipsum/gl-css/build/gl.min.css">
<script defer src="https://cdn.jsdelivr.net/gh/Aryqs-Ipsum/gl-css/build/gl.min.js"></script>

Or download build assets on github repository :

https://github.com/Aryqs-Ipsum/gl-css/tree/main/build

buttons

Buttons communicate actions that users can take. They are typically placed throughout your UI, in places like:

  • Dialogs
  • Modal windows
  • Forms
  • Cards
  • Toolbars

types

<button class="btn icon"><i class="material-icons">edit</i></button>

Styles

<button class="btn">flat</button>
<button class="btn outlined">outlined</button>
<button class="btn primary">primary</button>
<button class="btn secondary">secondary</button>

Sizes

<button class="btn outlined small">small</button>
<button class="btn outlined">medium</button>
<button class="btn outlined large">large</button>

Also work with "a" tag

Checkbox

Checkboxes allow users to select one or more items from a set. Checkboxes can turn an option on or off.

<label class="checkbox">
    <input type="checkbox">
    <span>Checkbox</span>
</label>
<label class="checkbox">
    <input type="checkbox" checked>
    <span>Checked checkbox</span>
</label>
<label class="checkbox">
    <input type="checkbox" disabled>
    <span>Disabled checkbox</span>
</label>

Drawer

Navigation drawers are recommended for:

  • Apps with five or more top-level destinations
  • Apps with two or more levels of navigation hierarchy
  • Quick navigation between unrelated destinations
<!-- menu button -->
<button class="btn icon js-drawer-trigger"><i class="material-icons">menu</i></button>

<!-- drawer -->
<aside class="drawer open">
    <div role="button" class="item active">
        <i class="material-icons">lightbulb</i>
        <span>Introduction</span>
    </div>
    <div role="button" class="item">
        <i class="material-icons">download</i>
        <span>Get Started</span>
    </div>
    <hr>
    <div role="button" class="item">
        <i class="material-icons">palette</i>
        <span>Colors</span>
    </div>
    <a href="#!" class="bottom">link</a>
</aside>
<div class="drawer-backdrop js-drawer-closable"></div>

<!-- app content -->
<main></main>

Loaders

Progress indicators inform users about the status of ongoing processes, such as loading an app, submitting a form, or saving updates. They communicate an app’s state and indicate available actions, such as whether users can navigate away from the current screen.

Circular

<div class="js-circular-loader"></div>

<!-- or this (if loader is needed before gl.js load) -->
<svg class="loader" width="42" height="42" xmlns="http://www.w3.org/2000/svg">
    <g>
        <ellipse ry="16" rx="16" cy="21" cx="21" stroke-width="4" fill="none"/>
    </g>
</svg>

Linear

Indeterminate

<div class="linear-loader indeterminate"></div>

Determinate

<div class="linear-loader">
    <div class="determinate" style="width: 25%;"></div>
</div>

Top app bar

The top app bar provides content and actions related to the current screen. It’s used for branding, screen titles, navigation, and actions. It can transform into a contextual action bar.

<header class="top-app-bar fixed elevated colored">
    <div class="wrapper">
        <div class="left">
            <button class="btn icon js-drawer-trigger"><i class="material-icons">menu</i></button>
        </div>
        <span class="brand">gl css</span>
        <div class="toolbar">
            <a href="#!" class="btn icon"><i class="material-icons">settings</i></a>
        </div>
    </div>
</header>
