Quickstart

Use assets with CDN link :

<!-- Material Design icons -->
<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">

<!-- gl-css files -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Aryqs-Ipsum/gl-css/build/gl.min.css">
<script defer src="https://cdn.jsdelivr.net/gh/Aryqs-Ipsum/gl-css/build/gl.min.js"></script>

Or download build assets on github repository :

https://github.com/Aryqs-Ipsum/gl-css/tree/main/build

buttons

Buttons communicate actions that users can take. They are typically placed throughout your UI, in places like:

  • Dialogs
  • Modal windows
  • Forms
  • Cards
  • Toolbars

types

<button class="btn icon"><i class="material-icons">edit</i></button>

Styles

<button class="btn">flat</button>
<button class="btn outlined">outlined</button>
<button class="btn primary">primary</button>
<button class="btn secondary">secondary</button>

Sizes

<button class="btn outlined small">small</button>
<button class="btn outlined">medium</button>
<button class="btn outlined large">large</button>

Also work with "a" tag
